# element-ui-electron

Element-UI issue reproduction: Tooltip doesn't work on Electron.js, but Popover does

#### Build Setup

``` bash
git clone https://github.com/antarasi/element-ui-electron.git
cd element-ui-electron
yarn
yarn run dev
```
